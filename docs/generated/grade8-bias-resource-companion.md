# Grade 8 Bias and Decision-Making — Resource Companion

This companion file sits beside the stable-core package and lists suggested printable resources for the five-day unit.

## Suggested print resources

### Day 1
- quick statement sort cards: fact, assumption, stereotype, opinion
- snap judgment image or profile cards

### Day 2
- media framing image set
- short caption and headline comparison cards

### Day 3
- recommendation critique cards
- app or quiz prompt examples

### Day 4
- cognitive bias scenario cards
- matching cards for confirmation bias, halo effect, bandwagon effect, availability bias

### Day 5
- bias pause cards
- optional classroom poster with the four-step routine: Notice, Question, Check, Choose

## Design notes
- keep print resources student-safe and broad
- avoid requiring identity disclosure
- use short, readable prompts that fit Grades 6–8
- keep the visual style aligned with Mr. Friess engine outputs
