# Documentation

This directory holds operating documents for the Mr. Friess Classroom Engine.

## Core operating docs

- [`house-workflow.md`](./house-workflow.md) — named workflow stages, handoff rules, and issue-triage model
- [`review-protocol.md`](./review-protocol.md) — operational review routine for lesson packages, rendered artifacts, and pull requests

## When to use these docs

Use `house-workflow.md` when you need the big-picture pipeline and handoff model.

Use `review-protocol.md` when you are actively reviewing:
- a lesson package
- a rendered artifact
- a docs/content PR
- a render/QA PR

## Working rule

Classify the problem first, then make the smallest safe fix in the correct layer.
