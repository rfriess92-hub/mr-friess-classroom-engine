# Math 8 Target Cards

Use these cards to start future Math 8 packages from target and evidence before context.

## 1. Fraction Sense and Operations

Students need to represent fractions visually and on number lines, compare fractions, use operations in simple measurement contexts, and explain if an answer is reasonable.

Best evidence: visual models, number-line work, operation practice, short measurement problem, reasonableness explanation.

Support moves: friendly denominators, fraction bars, one operation per page, partially completed examples, oral explanation.

Possible contexts: food/recipes, design/build measurement, consumer amounts, health choices, garden measurement.

## 2. Ratios, Rates, and Percents

Students need to represent ratios, use equivalent ratios, calculate unit rates, convert fractions/decimals/percents, and explain fair comparisons.

Best evidence: ratio table, unit rate table, percent benchmark organizer, best-buy or comparison task.

Support moves: visual ratio models, partial tables, friendly numbers, calculator-permitted prompts, two-choice comparison.

Possible contexts: consumer/money, growth rates, design quantities, health/body choices, digital/data.

## 3. Financial Literacy and Best Buy Decisions

Students need to calculate unit price, compare prices and amounts, calculate simple discounts, and decide best value.

Best evidence: unit price table, discount frame, best-buy challenge, recommendation using math evidence.

Support moves: two-product comparison, formula box, partial unit price table, visual price tags, calculator allowed.

Possible contexts: consumer/money, food systems, entrepreneurship, Careers, ADST materials.

## 4. Linear Relations and Growth Patterns

Students need to complete tables, identify start and rate, graph ordered pairs, use expressions, substitute, predict, and explain context.

Best evidence: table, graph, expression meaning organizer, prediction frame.

Support moves: one simple pattern, partial table, given expression, pre-labelled axes, sentence frames.

Possible contexts: garden growth, savings, fitness tracking, design scaling, digital/data.

## 5. Expressions, Substitution, and Two-Step Equations

Students need to understand variables, substitute values, simplify expressions, solve simple two-step equations, and explain what the solution means.

Best evidence: substitution sheet, equation-solving practice, meaning frame, correction page.

Support moves: formula box, one step at a time, colour-coded inverse operations, partial examples, calculator where appropriate.

Possible contexts: consumer costs, design formulas, digital rules, growth patterns, trades measurement.

## 6. Surface Area, Volume, and 3D Measurement

Students need to identify dimensions, choose formulas, calculate surface area or volume, use units, and explain what the answer measures.

Best evidence: labelled 3D diagram, calculation organizer, material/space estimate, explanation.

Support moves: pre-labelled diagrams, formula box, one shape at a time, friendly numbers, check-in boxes.

Possible contexts: planter boxes, packaging, storage, trades, design/build, food systems.

## 7. Pythagorean Theorem and Layout Checking

Students need to find a right triangle, label legs and hypotenuse, calculate a missing diagonal, compare measurements, and make a layout decision.

Best evidence: labelled diagram, calculation sheet, layout check table, conclusion frame.

Support moves: friendly-number example, pre-labelled diagram, formula box, partial calculation, calculator allowed.

Possible contexts: design/build/trades, garden layout, outdoor layout, technical plans.

## 8. Scale Drawings, Views, and Nets

Students need to use scale, convert real to drawing measurements, make top/front/side views, and communicate a design clearly.

Best evidence: scale conversion table, labelled technical plan, view page, explanation.

Support moves: fixed dimensions, given scale, partial conversion table, pre-started grid, top-view-only option.

Possible contexts: design/build/trades, garden design, maps, ADST drafting, community planning.

## 9. Data, Central Tendency, and Interpretation

Students need to organize data, calculate mean/median/range, create graphs, identify trends, and write conclusions from evidence.

Best evidence: data table, graph, calculation page, conclusion frame.

Support moves: simplified data set, fewer points, pre-labelled graph axes, calculation examples, sentence frames.

Possible contexts: garden growth, health choices, digital/data, consumer surveys, local environment.

## 10. Probability and Fair Games

Students need to describe likelihood, calculate simple probability, compare theoretical and experimental results, and decide if a game is fair.

Best evidence: probability table, spinner/dice experiment, fairness explanation.

Support moves: visual probability models, fewer outcomes, pre-filled tables, oral explanation.

Possible contexts: games, consumer promotions, digital simulations, classroom decision-making.

## 11. Math Skills Recovery and Numeracy Routines

Students need to rebuild number sense, operations, estimation, vocabulary, and confidence through short routines.

Best evidence: daily warm-ups, correction logs, strategy reflections, mini-checks.

Support moves: predictable layout, fewer questions, visual supports, oral rehearsal, success tracking.

Possible contexts: any low-clutter practical context.
