# Grade 8 Curriculum Target Card Bank

Purpose: keep future Grade 8 packages anchored to curriculum targets before context and artifact production.

Use this sequence:

1. Choose the curriculum target.
2. Name the student-friendly learning goal.
3. Choose one main context family.
4. Define the student product.
5. Define assessment evidence.
6. Define K-support / reduced-load access moves that preserve the same learning goal.
7. Separate teacher-facing and student-facing artifacts.

Context families:

- Garden / Planter / Food Systems
- Design / Build / Trades
- Community / Kamloops / Local Work
- Outdoor Place / Land / Environment
- Health / Body / Choices
- Digital / Coding / Data
- Consumer / Money / Market
- Stories / Identity / Growth

Guardrail: garden / planter / food systems is one option, not the default.
