# A1 Render Gap Report

## Current conclusion

The current engine render is not an acceptable A1 unit package.

It is a narrow renderer proof that produces:

- one teacher guide PDF
- one student sheet PDF
- one slide deck PPTX
- one exit ticket PDF
- rendered package QA JSON

That is not the actual Psychology 11/12 A1 package structure.

## Required A1 package structure

A1, Psychology as a Science, is a ten-day unit package. The complete package requires:

1. A1 Teacher Binder
2. A1 Student Packet
3. A1 Daily Slide Decks
4. A1 Assessment Pack
5. A1 Safety and Source Pack

The student packet must include:

- Vocabulary Organizer
- Handout 1 — Psychology Claim Sort
- Handout 2 — Make It Measurable
- Handout 3 — Research Method Match
- Handout 4 — The Causation Trap
- Handout 5 — Ethics Check
- Handout 6 — Reading: How Psychologists Study Behaviour
- Handout 7 — Claim Detective Practice
- Handout 8 — Research Design Challenge
- Handout 9 — Assessment Prep: Model Response Sort
- Handout 10 — Final Assessment Planning Sheet
- A1 Final Assessment — Flawed Claim Detective
- Exit Tickets for Days 1–10

The teacher binder must include:

- unit overview
- 10-day teacher sequence
- source alignment
- daily timing and teacher moves
- differentiation and tiering notes
- assessment look-fors
- model response calibration
- source/excerpt tracker
- teacher marking notes
- safety and boundary language

## Engine gap

The current engine output model is too narrow for this package. It can render the standard classroom bundle outputs, but does not yet preserve the full A1 source package structure.

Missing or under-supported areas:

- complete teacher binder document
- complete student packet document
- separate assessment pack
- rubric and model response bank
- safety redirect script bank
- sourced excerpt bank
- editable DOCX or equivalent source exports
- daily slide deck bundle for all ten days
- package-level completeness QA

## QA decision

A1 must remain blocked until these gaps are resolved. A green render of a daily proof is not a complete A1 unit package.
