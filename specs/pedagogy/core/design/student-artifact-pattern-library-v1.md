# Student Artifact Pattern Library v1

Purpose: define richer student-facing artifact patterns so lessons do not default to generic blank-line worksheets when stronger structures are more instructionally appropriate.

## Design rule

The student artifact should fit the thinking task.

Do not default to long open lines when the lesson really needs:
- sorting
- matching
- ranking
- claim/evidence organization
- scenario analysis
- planning structure

## Recommended student artifact patterns

### `classification_table`
Use when students must sort examples into categories or clusters.

Best for:
- career clusters
- science classification
- text/theme sorting
- source sorting

### `match_and_justify`
Use when students match items and explain their reasoning.

Best for:
- examples to categories
- cause to effect
- role to responsibility

### `claim_evidence_box`
Use when students must state a claim and support it with evidence.

Best for:
- ELA
- social studies
- careers reflection with evidence

### `ranking_grid`
Use when students compare options and justify relative strength or fit.

Best for:
- best fit / least fit
- strongest evidence
- pathway comparisons

### `scenario_response_panel`
Use when students respond to a short case or scenario.

Best for:
- careers
- health
- decision-making
- applied reasoning

### `reflection_ladder`
Use when students move from recall to personal connection to forward-looking reflection.

Best for:
- exit work
- reflection sheets
- metacognitive closure

### `planning_frame`
Use when students need structure before final writing.

Best for:
- final response preparation
- inquiry planning
- project planning

### `choice_board_response`
Use when students should choose among response pathways while still producing comparable evidence.

Best for:
- differentiated response
- interest-based choice
- moderate personalization

## Pattern guidance by evidence type

### If final evidence is short reflection
Prefer:
- `reflection_ladder`
- short structured exit ticket

### If final evidence is classification or sorting
Prefer:
- `classification_table`
- `match_and_justify`

### If final evidence is written justification
Prefer:
- `claim_evidence_box`
- `scenario_response_panel`

### If final evidence is a longer final response
Prefer:
- `planning_frame` first
- then `final_response_sheet`

## Constraints

Student artifact patterns must not:
- leak teacher-only guidance
- hide the evidence location
- create ambiguous final evidence placement
- overcomplicate a short lesson with unnecessary structure

## Minimum rule for v1

Choose the simplest structure that supports the task clearly.

## First recommended upgrade target

Upgrade `Careers 8 — Exploring Career Clusters` by replacing the generic worksheet shape with:
- `classification_table` for the sorting task
- `match_and_justify` or `claim_evidence_box` for reasoning
- `reflection_ladder` structure for the final reflection prompt
