# Classroom Content Contract

This contract protects classroom usefulness, BC alignment, and teacher/student separation.

## Core rule

Classroom content must be usable by Mr. Friess without manual reconstruction. It should sound like clear teacher-facing classroom material, not generic filler.

## BC alignment

Content phases should make curriculum alignment visible through subject and grade context, curricular competencies or skill focus where appropriate, real-world classroom connection, assessment language that matches the task, and enough scaffolding for K designation and diverse learners without lowering the core goal.

## Differentiation rule

Differentiation must stay aligned. Reduced-output, supported, visual, oral, partner, and extension options should still point toward the same learning target.

## Teacher/student separation

Teacher-facing materials may include pacing notes, setup instructions, facilitation moves, assessment guidance, likely misconceptions, answer keys, and intervention suggestions.

Student-facing materials may include learning targets, clear task steps, examples and sentence frames, reduced-response options, checklists, reflection prompts, and self-assessment.

Student-facing materials must not include answer keys, private teacher notes, implementation metadata, grading rationales meant only for the teacher, repo instructions, or agent instructions.

## Classroom realism

Prefer concrete local and classroom details when known. For this project, Kamloops, BC can be used when a local reference is useful and appropriate.

## Minimum content quality bar

A generated classroom package should include clear purpose, materials or setup when needed, student-facing instructions, assessment or completion evidence, scaffolded access points, extension or challenge option when useful, and teacher notes separated from student handouts.
