# Psychology 11/12 Lens-Based Cycle Architecture

## 1. Purpose

This document consolidates the current planning decision for Psychology 11/12 and serves as the conceptual companion to the A-F Psychology Lens Matrix. It is intended to prevent drift, preserve the conceptual architecture, and act as a source-of-truth for future course maps, semester plans, student packets, teacher binders, assessments, and slide decks.

The working decision is that Psychology 11/12 should not be a rigid textbook march and should not become a random topic grab bag. It should be a lens-based spiral curriculum: each semester revisits the major domains of psychology from a different organizing point of view.

The detailed planning matrix lives in `source/psychology_11_12/course/lens_matrix.md`. This reference file should stay focused on the architecture, rules, safety boundaries, and source-alignment logic.

## 2. Comprehension Scan

### What we are not building

- Not a rigid textbook sequence where each semester is only one content block.
- Not a true grab bag where topics are chosen randomly or only by interest.
- Not a therapy, diagnosis, personal-disclosure, or peer-counselling course.
- Not a course that treats psychology as advice, opinion, personality quizzes, or pop psychology.

### What we are building

- A lens-based spiral curriculum for Psychology 11/12.
- Six semester lenses, A-F, each with a clear point of view.
- A structured module menu inside each cycle so the course stays flexible but coherent.
- A repeated psychology domain map so brain/body, learning, memory, cognition, development, identity, social psychology, mental health, ethics, and applied psychology recur across the program.
- A classroom-safe, evidence-focused senior psychology course grounded in responsible claim analysis and careful language.

## 3. Core Architecture Decision

The final architecture should be six semester lenses plus permanent through-lines.

The lenses organize semesters. The through-lines keep the course psychologically complete, culturally aware, developmentally aware, biologically grounded, evidence-based, and ethically safe.

| Element | Decision | Reason |
|---|---|---|
| Cycles A-F | Keep as semester-level lenses. | Gives each semester a clear identity and prevents random topic drift. |
| Grab-bag topics | Use as structured module menus inside each lens. | Preserves flexibility without losing coherence, coverage, or assessment logic. |
| Cultural lens | Use as a required through-line, not a separate cycle for now. | Culture should shape every psychology question rather than appear as one isolated unit. |
| Developmental lens | Use as a required through-line. | Students should regularly ask how concepts change across age, experience, and life stage. |
| Biological lens | Use as a required through-line. | Each cycle should connect mind, behaviour, brain, body, nervous system, hormones, stress, sleep, and biology where appropriate. |
| Ethics/evidence lens | Use as the disciplinary spine across every cycle. | Keeps the course academic, safe, and psychologically responsible. |

## 4. Six Semester Lenses

| Cycle | Lens Name | Core Question | Primary Function |
|---|---|---|---|
| A | Scientific / Evidence | How do psychologists know what they claim to know? | Builds disciplinary habits: evidence, methods, ethics, limits, and careful claim analysis. |
| B | Intellectual / Learning | How do humans learn, think, create, and adapt? | Explores learning, memory, neuroplasticity, cognition, creativity, education, motivation, and skill development. |
| C | Relational / Identity | How are humans shaped by connection, belonging, sexuality, personality, and social context? | Explores the social brain, relational memory, personality, identity, human sexuality, relationships, belonging, and social influence. |
| D | Wellness / Change | How do people cope, struggle, change, and access support? | Explores stress, coping, mental health literacy, treatment, support systems, resilience, and behaviour change. |
| E | Justice / Responsibility | How should psychology be used when harm, safety, memory, law, and accountability are involved? | Explores memory, bias, legal psychology, risk, responsibility, mental health and law, forensic ethics, and justice systems. |
| F | Systems / Media / Influence | How are humans shaped by designed environments, media, persuasion, and technology? | Explores attention, habit design, misinformation, persuasion, consumer psychology, social media, AI, and ethical design. |

## 5. Permanent Through-Lines

These are not separate units. They are recurring questions that should appear inside every cycle, every major unit, and every capstone.

| Through-Line | Recurring Question | Use in Planning |
|---|---|---|
| Cultural / Contextual | How do culture, community, identity, language, power, family, environment, and access shape this? | Prevents treating one group as the default human norm. Supports culturally responsive and context-aware psychology. |
| Developmental / Lifespan | How does this change across childhood, adolescence, adulthood, and older adulthood? | Keeps students from assuming a concept works the same way across all ages and stages. |
| Biological / Brain-Body | What is happening in the brain, body, nervous system, endocrine system, or biology, and what are the limits of that explanation? | Grounds psychology in body systems without reducing people to biology. |
| Ethical / Evidence | How do we know? What are the limits? What responsibilities come with this knowledge? | Preserves the science-and-safety spine of the course. |

## 6. Required Psychology Domains

Each semester should touch the same broad psychology domains. The depth changes by lens, but the domains should recur so the course feels like a full psychology program rather than isolated topic packets.

| Domain | Must Appear? | Depth Rule |
|---|---:|---|
| Psychology as science / evidence | Yes | Medium every cycle; highest in Cycle A. |
| Brain, body, and behaviour | Yes | Medium every cycle; adapted to the lens. |
| Learning and memory | Yes | Medium every cycle; highest in Cycle B and relevant to E and F. |
| Cognition, thinking, intelligence, bias | Yes | Medium every cycle; highest in Cycle B and F. |
| Development across the lifespan | Yes | Light to medium every cycle; must include adolescence when relevant. |
| Emotion, motivation, personality, identity | Yes | Light to medium every cycle; highest in Cycle C. |
| Social psychology and relationships | Yes | Medium every cycle; highest in Cycle C and E. |
| Stress, health, and mental health literacy | Yes | Light to medium every cycle; highest in Cycle D. |
| Applied psychology | Yes | Medium every cycle; capstone link preferred. |
| Ethics and responsible use | Yes | Medium every cycle; embedded in assessments and discussion rules. |

## 7. How the Same Topic Changes by Lens

The same psychology topic can appear more than once because the point of view changes. This prevents repetition while strengthening transfer.

| Topic | Cycle A: Scientific / Evidence | Cycle B: Intellectual / Learning | Cycle C: Relational / Identity | Cycle D: Wellness / Change | Cycle E: Justice / Responsibility | Cycle F: Systems / Media / Influence |
|---|---|---|---|---|---|---|
| Memory | How do we study memory with evidence? | How does memory support learning and expertise? | How do relationships shape emotional and relational memory? | How does stress affect memory and coping? | How reliable is eyewitness testimony? | How do repetition, media, and misinformation shape memory? |
| Brain/body | How do psychologists connect brain, behaviour, and evidence? | How does neuroplasticity support learning? | What happens in the brain during connection, threat, attraction, and rejection? | How do stress systems affect wellbeing? | How do arousal, threat, and impulse control affect responsibility? | How do reward, novelty, and attention systems respond to designed environments? |
| Social psychology | How do social claims need evidence? | How do peers and expectations affect achievement? | How do belonging, norms, attraction, and identity shape behaviour? | How does social support buffer stress? | How do authority, bias, and groups affect legal decisions? | How do social proof, influencer culture, and algorithms shape behaviour? |

## 8. Operating Rules

1. Every cycle has one dominant lens.
2. Every cycle touches all major psychology domains, but not at equal depth.
3. Every unit checks culture/context, development, brain/body, evidence, and ethics.
4. Every cycle includes a capstone that requires students to synthesize multiple psychology domains through that cycle lens.
5. Student-facing work uses fictional scenarios, public examples, neutral case studies, source analysis, and low-risk academic reflection.
6. Teacher-facing materials may include answer keys, delivery notes, safety guidance, misconceptions, differentiation, and marking support.
7. No student-facing artifact should include teacher rationale, answer keys, hidden marking notes, or prompts that require private disclosure.
8. OpenStax Psychology 2e remains the academic spine, but material should be translated into classroom-appropriate lessons rather than reproduced directly.

## 9. Safety Boundary

The course must remain academic and classroom-safe. Students should learn to think like psychology students; they should not be trained to diagnose, counsel, disclose, or reduce people to labels.

| Do Not Require Students To | Use Instead |
|---|---|
| Disclose trauma, diagnoses, family history, private relationships, or identity struggles. | Fictional scenarios, public examples, neutral case studies, and low-risk academic reflection. |
| Diagnose themselves, diagnose peers, or analyze classmates. | Concept application, evidence evaluation, source analysis, and teacher-controlled discussion structures. |
| Role-play therapy, crisis counselling, or peer support beyond student boundaries. | Support literacy, referral awareness, careful language, and appropriate help-seeking concepts. |
| Turn forensic psychology into graphic true-crime entertainment. | Sanitized fictional cases, legal reasoning, memory reliability, bias analysis, and ethics. |
| Turn media psychology into moral panic or student shaming. | Agency, design awareness, persuasion analysis, evidence, and ethical design critique. |

## 10. Cycle-Specific Risk Controls

| Cycle | Specific Control |
|---|---|
| A | Lower risk, but still avoid diagnosis, unsupported interpretation, and overclaiming. |
| B | Avoid manipulative behaviour-control framing. Use study habits, routines, skill learning, sports practice, workplace training, and fictional examples. |
| C | Higher disclosure risk. Avoid personal attachment-style sharing, relationship-history sharing, family-conflict disclosure, and private identity analysis. |
| D | Highest safety-control requirement. Do not ask for diagnoses, trauma disclosure, self-harm history, or therapy role-play. Use fictional scenarios and general support literacy. |
| E | Avoid graphic crime detail and true-crime entertainment. Use sanitized or fictional cases. |
| F | Avoid moral panic about technology. Teach analysis, agency, evidence, and design awareness rather than shame. |

## 11. Assessment Spine

Assessment should repeat across cycles so students understand the intellectual routine of psychology. The content changes by lens; the reasoning pattern remains stable.

| Assessment Layer | Purpose |
|---|---|
| Vocabulary and concept checks | Build accurate disciplinary language. |
| Scenario applications | Apply concepts to fictional or neutral examples. |
| Source/evidence analysis | Distinguish evidence, opinion, claim, inference, correlation, causation, and limits. |
| Short quizzes/tests | Check foundational knowledge without making memorization the only pathway. |
| Applied mini-project | Use psychology responsibly in a realistic setting. |
| Capstone/performance task | Synthesize domains through the cycle lens. |
| Teacher marking guide | Support consistent assessment, look-fors, differentiation, and feedback language. |

## 12. Artifact Model

Each cycle should eventually be able to produce the same artifact family. This keeps future development organized and prevents student-facing and teacher-facing material from mixing.

| Artifact | Audience | Notes |
|---|---|---|
| Teacher binder | Teacher | Delivery guide, safety notes, misconceptions, differentiation, answer keys where appropriate. |
| Student packet | Student | No answer keys or teacher-only rationale. |
| Assessment pack | Student | Student-facing assessment tasks only. |
| Source sheet | Student | Short, purposeful, traceable support for concepts and vocabulary. |
| Capstone packet | Student | Synthesis task for the cycle lens. |
| Slide deck | Student | Low-density, classroom-safe, student-facing language. |
| Marking guide | Teacher | Rubrics, look-fors, feedback language, accommodation notes. |
| QA bundle | Teacher/repo | Render checks, answer-leakage checks, visibility checks, known issues, artifact inventory. |

## 13. Next Step: Course Map From the Lens Matrix

The A-F Psychology Lens Matrix now exists as the next planning control panel. The next source artifact should be `course_map.md`, which translates the architecture and matrix into a stable program map.

Recommended `course_map.md` structure:

| Section | Purpose |
|---|---|
| Program model | Explain the six-lens spiral curriculum. |
| Cycle overview table | Summarize each cycle's lens, essential question, core domains, and capstone. |
| Three-year rotation | Show how A-F can operate as a multi-year non-repeating sequence. |
| Semester template | Provide the repeated unit rhythm used across cycles. |
| Assessment spine | Connect vocabulary, scenario application, source analysis, projects, and capstone work. |
| Safety model | Preserve the no-diagnosis, no-therapy, no-forced-disclosure boundary. |
| Artifact pathway | Identify which files should be produced next for each cycle. |

Course-map design rule:

Use the matrix to decide what belongs in each cycle. Do not turn the matrix into six overloaded semester plans. Each semester should select priority modules from the matrix while preserving the dominant lens, required through-lines, and capstone direction.

## 14. Source Alignment Notes

- Academic spine: OpenStax Psychology 2e.
- Course stance: concept-based, classroom-safe, evidence-focused senior secondary psychology instruction.
- Core student habit: define concepts accurately; distinguish evidence from opinion; separate observation, inference, claim, and evidence; recognize uncertainty and limits; avoid self-diagnosis and amateur peer diagnosis.
- Whole-course domains: psychology as science; brain/body; consciousness/perception; learning/memory; cognition; development; emotion/motivation/personality; social psychology/relationships; stress/mental health; applied psychology.
- Cycle A-F source alignment remains useful, but the new model treats these as lenses rather than isolated content bins.
