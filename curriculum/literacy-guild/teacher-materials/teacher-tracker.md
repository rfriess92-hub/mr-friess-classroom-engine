# Teacher Tracker

## Purpose

The teacher tracker keeps the Literacy Guild manageable.

It should help the teacher see:

- current path goals
- evidence submitted
- likely rank-ups
- next teacher-table moves
- progress check dates
- students needing follow-up

## Weekly review

At the end of the week, check:

- Who submitted solo evidence?
- Who needs a conference?
- Who may be ready for a rank-up?
- Which path needs reteaching?
- Which students need a lower-friction entry task next week?

## Workload guardrail

Do not track everything.

Track only what changes instruction:

- current goal
- submitted evidence
- next move
- conference need

If the tracker takes longer than the feedback, simplify it.
