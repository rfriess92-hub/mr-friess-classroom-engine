# Literacy Guild Progress Model

## Purpose

DIBELS is used as a private progress check and teacher planning tool.

It does not become:

- a public score
- a leaderboard
- a character level
- a rank by itself
- a grade

Assessment informs the game. It is not the game.

## Current DIBELS 8 notes

DIBELS 8 is designed for kindergarten through Grade 8.

For this course, that means:

- Grade 8 students can use the tool within its intended grade range.
- Grade 9-10 students may use Grade 8 forms for instructional progress information, but this should be treated carefully.
- Off-grade results should guide instruction, not label students.

## Path connections

| Literacy Guild path | Teacher anchor | Progress source |
|---|---|---|
| Voice | Fluency | ORF, fluency recordings, repeated reading notes |
| Insight | Comprehension | Maze, written responses, oral explanations, gist/evidence tasks |
| Wordcraft | Vocabulary | Teacher-created vocabulary evidence, morphology tasks, word-use checks |

## Voice and ORF

ORF connects most directly to Voice.

Use ORF information to notice:

- accuracy
- rate
- self-correction
- hesitation patterns
- phrasing needs
- growth from baseline

Do not use ORF to publicly rank students.

Student-facing reflection prompts:

- What got smoother?
- Where did I stop or slow down?
- What word or sentence pattern was difficult?
- What should I practise next?

## Insight and Maze

Maze connects to comprehension, but it is not the whole picture of comprehension.

Use Maze information with classroom evidence:

- gist statements
- evidence-based answers
- inference explanations
- discussion notes
- comprehension repair tasks

Student-facing reflection prompts:

- Did I slow down enough to make meaning?
- Did I reread before choosing?
- Which choice made sense in the sentence?
- What strategy should I use next time?

## Wordcraft evidence

DIBELS does not give a separate vocabulary path score for this course model.

Wordcraft needs classroom evidence.

Useful Wordcraft evidence:

- word collection from a text
- context clue explanation
- prefix, suffix, root, or base-word breakdown
- word map
- sentence using a target word correctly
- oral explanation of a word strategy
- revision using more precise vocabulary

## Progress check cycle

Suggested cycle:

| Timing | Action | Purpose |
|---|---|---|
| Start of course | Baseline progress check | Set private starting goals |
| Weekly | Classroom evidence | Track Voice, Insight, and Wordcraft practice |
| Every 2-3 weeks | Fluency check or recording | Monitor Voice growth |
| Monthly | Maze or comprehension check | Monitor Insight growth |
| Every 6-8 weeks | Progress conference | Review evidence and set next goals |
| End of term | Growth snapshot | Summarize progress and next steps |

## Student progress conference

A short conference should answer:

1. What is getting stronger?
2. What evidence shows that?
3. What is still hard?
4. What strategy should come next?
5. Which path should be the next focus?

## Game connection

DIBELS can trigger:

- a new personal goal
- a teacher-table group
- a strategy recommendation
- a reflection prompt
- Growth XP for improvement from baseline

DIBELS should not directly trigger:

- public level changes
- public rank labels
- public comparison
- loss of XP

## Teacher tracker fields

Recommended tracker fields:

- student name
- current Voice goal
- current Insight goal
- current Wordcraft goal
- latest ORF note
- latest Maze note
- vocabulary evidence note
- next teacher-table move
- conference date
- next check date

## Student-facing language

Use progress check language, not failure language.

Suggested script:

This check helps us see what is getting stronger and what needs training next. It is not a public score. We use it to choose better strategies and better practice.

## Main rule

DIBELS helps the teacher and student choose the next training move. It does not define the student, the character, or the public game status.
