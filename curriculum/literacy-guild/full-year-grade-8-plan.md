# Full-Year Grade 8 Implementation Plan

## Purpose

This version is for a full academic year. It should build routine, belonging, reading confidence, and long-term growth.

## Launch stance

Students are new to the school. The system can be introduced as the normal way this class works.

## Campaign map

### September: Enter the Guild

- class identity
- three paths
- routines
- private baseline checks
- first short quest

### October-November: Voice Trial

- fluency routines
- repeated reading
- phrasing
- accuracy
- confidence

### December-January: Word Vault

- vocabulary routines
- context clues
- prefixes and suffixes
- roots
- word use

### February-March: Insight Gate

- gist
- evidence
- inference
- comprehension repair
- written and oral explanation

### April-May: Three-Path Campaign

- integrated weekly quests
- longer evidence tasks
- stronger independence
- party challenges

### June: Final Guild Portfolio

- growth reflection
- evidence review
- student conference
- next-step planning

## Design note

The full-year version can include more character development and a fuller campaign, but mechanics should still be introduced slowly.
