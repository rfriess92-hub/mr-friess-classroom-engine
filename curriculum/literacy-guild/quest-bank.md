# Literacy Guild Quest Bank

## Purpose

This file lists the reusable quests available in the Literacy Guild.

The quest bank supports controlled student choice. Students may choose route, order, and format where appropriate, but all required skill destinations must eventually be completed.

## Quest bank rules

1. Required core quests cannot be skipped.
2. Choice quests must assess the same target skill.
3. Side quests add practice, XP, title evidence, or ability evidence.
4. Recovery quests are support routes, not punishment.
5. Boss quests require core quest completion.
6. The Game Master may assign or unlock quests based on evidence.

## Quest fields

Each quest should include:

- quest name
- arc
- type
- main path
- support paths
- ability connection
- required evidence
- unlock rule
- teacher notes

## Arc 1: Enter the Guild

### Enter the Guild

Type: Core Quest

Main path: All paths

Purpose: Introduce the course frame, dignity rules, and Guild identity.

Evidence:

- Guild Passport started
- exit ticket on reading strengths and needs

Unlock rule: Available at course start.

### Learn the Loop

Type: Core Quest

Main path: All paths

Evidence:

- revised short task

- path sort

Unlock rule: Available after three paths are introduced.

### First Evidence

Main path: All paths

Purpose: Submit first low-stakes evidence.

Evidence:

- one solo evidence item

Unlock rule: Available after Learn the Loop and Path Finder.

## Teacher note

The quest bank is not a script. It is a menu of structured destinations.

Use it to build weekly lessons, intervention groups, and student choice maps without losing required skill coverage.
