# Weekly Quest Template

## Purpose

A quest is the weekly container for Literacy Guild instruction.

Each quest should include all three paths, but one path may be the main focus.

## Quest header

- Quest title:
- Week:
- Class version: full-year / semester
- Main path: Voice / Insight / Wordcraft
- Support paths:
- Text or text set:
- Teacher table focus:

## Student-facing hook

Write one or two sentences that make the task feel purposeful.

Keep the hook light. The reading task must stay clear.

Example:

This week, your party is entering a locked archive. To move forward, you need to read carefully, unlock key words, and prove what the text is really saying.

## Learning target

Students will be able to:

- read or reread a selected text section with a clear strategy
- explain the meaning of the text using evidence
- unlock and use important words from the text

## Path tasks

### Voice task

Fluency practice connected to the text.

Options:

- echo read
- partner reread
- phrase-cued reread
- punctuation practice
- fluency recording
- personal-best reread

Evidence:

- reread note
- teacher observation
- recording
- self-reflection

### Insight task

Comprehension work connected to the text.

Options:

- gist statement
- evidence-based answer
- inference question
- summary
- comprehension repair
- discussion response

Evidence:

- written response
- oral explanation
- highlighted evidence
- revised answer

### Wordcraft task

Vocabulary work connected to the text.

Options:

- collect key words
- context clue explanation
- prefix/suffix/root work
- word map
- precise word use
- sentence revision

Evidence:

- word breakdown
- word map
- sentence using the word
- oral explanation

## Daily flow

| Day | Quest phase | Main work | Evidence |
|---|---|---|---|
| Monday | Launch | Introduce text, hook, target, and key words | Entry reflection |
| Tuesday | Train | Model and practise main skill | Guided practice |
| Wednesday | Party challenge | Use skill with support | Group notes or oral check |
| Thursday | Evidence build | Complete solo evidence task | Draft or recording |
| Friday | Boss / level-up | Submit evidence and reflect | Final evidence and reflection |

## Solo evidence

Every quest needs one piece of individual evidence.

Solo evidence options:

- one supported answer
- one revised answer
- one fluency recording and reflection
- one word strategy explanation
- one short paragraph
- one teacher conference note

## Party support

Party work may support the quest, but it should not replace solo evidence.

Useful party roles:

- Voicekeeper
- Insight Keeper
- Wordsmith
- Evidence Hunter
- Quest Captain

## XP menu for the quest

Choose only a few XP actions each week.

| Action | XP |
|---|---:|
| Complete training task | 10 |
| Use named strategy | 15 |
| Improve after feedback | 15 |
| Submit clear evidence | 20 |
| Complete reflection | 10 |
| Show growth from baseline | 20 |
| Support a peer appropriately | 10 |

## Reflection prompt

Choose one:

- What got stronger this week?
- What strategy helped you most?
- What evidence shows progress?
- What should you train next?

## Teacher notes

- Keep texts short enough for repeated reading and close work.
- Do not overload the quest with too many products.
- One strong piece of evidence is better than several weak worksheets.
- If the week is shortened, protect the training loop and solo evidence.
