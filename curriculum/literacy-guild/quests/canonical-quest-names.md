# Canonical Quest Names

## Purpose

This file is the source of truth for quest and arc names before printable packet generation.

Use these names in packets, trackers, teacher materials, student maps, and printable versions.

|---:|---|---|

## Naming rules

Use `Quest` in the title when the item is a student task.

Use `Arc` only for multi-week instructional sequences.

Use `Boss Quest` for larger demonstrations after training.

Use `Recovery Quest` or `Reroute` for support routes, not punishment.

Use `Integrated Campaign`, not `Three-Path Campaign`, for the arc name.

Keep `Three-Path Boss Quest` as the boss task name inside the Integrated Campaign.
