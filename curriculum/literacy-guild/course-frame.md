# Literacy Guild Course Frame

## Course identity

The Literacy Guild is a reading growth system built on three paths:

- Voice
- Insight
- Wordcraft

## Teacher anchors

- Voice supports fluency.
- Insight supports comprehension.
- Wordcraft supports vocabulary.

## Student message

This class is a training space. The goal is not to prove who is already strong. The goal is to build evidence that each student is improving.

## Classroom model

The course uses whole-class instruction, small-group rotation, computer-supported practice, solo evidence, and private progress conferences.

## 79-minute class structure

1. Daily Encounter
2. Guild Lesson
3. Training Rotations
4. Solo Evidence
5. Character Log

## Two class versions

The full-year Grade 8 class uses a slower launch and longer campaign.

The semester cohort uses a shorter reset, simpler lore, and faster routines.
