# Game Master Role

## Purpose

This file defines the teacher-facing role in the Literacy Guild.

The teacher is the Game Master. This gives the classroom system a clear adult role without pretending the teacher is just another player.

## Core idea

The Game Master runs the world, protects the rules, gives feedback, and keeps the game connected to real reading growth.

The Game Master is not competing with students.

## Why this role matters

The Game Master role helps the teacher:

- frame quests
- control pacing
- protect student dignity
- keep assessment private
- approve rank-ups
- give feedback
- decide when to simplify mechanics
- connect the game layer to instruction

## Teacher character or not?

Yes, the teacher can have a character, but it should be different from student characters.

The teacher character should represent the teacher's classroom role, not a student-style level track.

Recommended teacher identity:

- Title: Game Master
- Optional name: Guildmaster, Archivist, Keeper, Guide, or Questmaster
- Role: designs quests, opens challenges, gives feedback, confirms evidence, protects the Guild

Avoid giving the teacher a student-style rank, XP total, or leaderboard position.

## Teacher abilities

Teacher abilities should be classroom moves, not powers that interrupt student agency.

| Ability | Meaning |
|---|---|
| Quest Call | Launches the learning challenge and names the target. |
| Strategy Reveal | Teaches or reminds students of a useful reading strategy. |
| Evidence Check | Confirms whether evidence is clear enough for progress. |
| Reroute | Adjusts a task when the class needs a different path. |
| Safe Table | Pulls a small group for protected teacher-table support. |
| Pause the Realm | Stops the game layer to reteach the real skill. |
| Level Gate | Approves rank-ups or holds them for more evidence. |
| Private Counsel | Holds a private progress conference. |
| Shield of Dignity | Keeps progress data private and prevents comparison. |

## Teacher title options

Choose one main title and use it consistently.

Recommended options:

| Title | Tone |
|---|---|
| Game Master | Clear D&D-inspired language; familiar to students who know RPGs. |
| Guildmaster | Strong classroom fit; less tied to one game system. |
| Questmaster | Directly connected to weekly quests. |
| Archivist | Good for literacy, texts, records, and evidence. |
| Keeper | Simple, mature, flexible. |

Recommendation: use **Guildmaster** or **Game Master**.

Game Master is clearer. Guildmaster is more original and classroom-safe.

## Suggested teacher frame

Student-facing script:

In this class, I am the Game Master. That means I design the quests, teach the strategies, check the evidence, and protect the rules. I do not rank you against each other. My job is to help each of you train, collect evidence, and know your next move.

## Game Master responsibilities

### Before class

- choose the quest target
- prepare text and supports
- decide which path is the focus
- prepare teacher-table moves
- choose possible XP actions
- decide what evidence will count

### During class

- launch the challenge
- teach the strategy
- watch for confusion
- give feedback
- protect solo evidence time
- keep the pace realistic
- reduce game mechanics if they distract from learning

### After class

- check evidence quickly
- update tracker only where needed
- note students for conference
- approve or delay rank-ups
- adjust next quest or small-group plan

## What the Game Master should not do

The Game Master should not:

- turn private data into public game status
- over-explain lore instead of teaching reading
- award points for everything
- let confident students dominate the story
- make students perform publicly before routines are safe
- confuse character level with reading level
- keep mechanics that create shame or comparison

## Teacher abilities as reminders

The teacher abilities can appear on a small reference card.

They are reminders for the teacher, not a second game system for students to manage.

## Student-facing rule

The Game Master can change the quest if the class needs a better route. That is not failure. It means the teacher is choosing the right training path.

## Design caution

Use the Game Master identity enough to create atmosphere, but not so much that it becomes performance. The teacher remains the teacher. The game role gives structure, language, and consistency.
