# Voice, Insight, and Wordcraft

## Purpose

This file defines the three Literacy Guild paths.

Teacher-facing anchors:

- Voice = Fluency
- Insight = Comprehension
- Wordcraft = Vocabulary

Student-facing promise:

You are training reading skills. You are not being ranked against other readers.

## Shared rank pattern

Each path uses five ranks.

| Rank | General meaning |
|---:|---|
| 1 | I can attempt the skill with support. |
| 2 | I can use the skill in a simple task. |
| 3 | I can use the skill with more independence. |
| 4 | I can apply the skill in a larger task. |
| 5 | I can explain or transfer the skill. |

## Voice

Teacher anchor: Fluency

Student meaning: I can read more smoothly, accurately, and confidently.

Instructional focus:

- accuracy
- rereading
- phrasing
- punctuation
- expression
- stamina
- self-correction

### Voice rank ladder

| Rank | Student target | Evidence examples |
|---:|---|---|
| 1 | I can reread a short passage with support. | Completes echo reading, choral reading, or guided rereading. |
| 2 | I can improve after practice. | Shows better accuracy, smoother pacing, or fewer stops on a second read. |
| 3 | I can use punctuation and phrasing. | Pauses at punctuation and chunks sentences into meaningful parts. |
| 4 | I can notice and fix some errors. | Stops, rereads, or self-corrects when meaning breaks. |
| 5 | I can explain a fluency strategy. | Names the strategy used and explains how it helped the reading. |

### Voice XP actions

- complete a reread with effort
- practise a selected passage twice
- use punctuation to improve phrasing
- record and reflect on a fluency read
- self-correct a misread word
- improve from a personal baseline

### Voice checkpoint names

- Echo Reader
- Second Read
- Phrase Builder
- Error Catcher
- Confident Voice

### Teacher notes

For students significantly below grade level, Voice work should use texts they can access with support. Do not make oral reading public unless the routine is safe and predictable. Growth from a student's own baseline matters more than comparison to peers.

## Insight

Teacher anchor: Comprehension

Student meaning: I can understand, explain, and prove my thinking.

Instructional focus:

- gist
- evidence
- inference
- questioning
- summarizing
- meaning repair
- discussion
- written explanation

### Insight rank ladder

| Rank | Student target | Evidence examples |
|---:|---|---|
| 1 | I can say what a short section is mostly about. | Gives a gist statement after reading or listening. |
| 2 | I can find a detail that supports my answer. | Points to or copies a useful text detail. |
| 3 | I can answer using evidence. | Writes or says an answer with a reason from the text. |
| 4 | I can explain an inference. | Explains what the text suggests and what clues support it. |
| 5 | I can choose a repair strategy when confused. | Rereads, asks a question, summarizes, or checks evidence to rebuild meaning. |

### Insight XP actions

- write a clear gist statement
- find useful evidence
- revise an answer with stronger support
- explain an inference
- ask a question that helps the group understand
- identify confusion and use a repair strategy

### Insight checkpoint names

- Gist Finder
- Evidence Tracker
- Inference Builder
- Meaning Repair
- Text Strategist

### Teacher notes

Insight work should not depend only on silent independent reading. Use read-aloud, audio, chunking, partner talk, and teacher table support when needed. The goal is to train comprehension thinking, not to punish slow or inaccurate decoding.

## Wordcraft

Teacher anchor: Vocabulary

Student meaning: I can unlock, understand, and use important words.

Instructional focus:

- word awareness
- context clues
- prefixes
- suffixes
- roots
- academic vocabulary
- word use in speech and writing

### Wordcraft rank ladder

| Rank | Student target | Evidence examples |
|---:|---|---|
| 1 | I can notice important or unknown words. | Marks words that matter or words not yet understood. |
| 2 | I can use context to make a smart guess. | Uses nearby words or sentences to infer meaning. |
| 3 | I can break words into meaningful parts. | Identifies a prefix, suffix, root, or base word. |
| 4 | I can use a new word correctly. | Uses the word in a sentence, explanation, or discussion. |
| 5 | I can teach a word strategy. | Explains context, morphology, or word use to another student. |

### Wordcraft XP actions

- collect important words from a text
- use context clues
- break a word into parts
- build a word map
- replace a vague word with a stronger word
- use a target word correctly in writing or speech

### Wordcraft checkpoint names

- Word Collector
- Context Reader
- Root Breaker
- Precision Upgrade
- Word Mentor

### Teacher notes

Wordcraft needs explicit teaching. Do not assume vocabulary will grow only through exposure. Use short, repeated routines for context, morphology, and word use. Vocabulary evidence should include using words, not just copying definitions.

## Path balance

A weekly quest should usually include all three paths, but one path can be the main focus.

Example:

- Main path: Insight
- Support path: Wordcraft
- Quick path: Voice

This prevents the course from becoming three separate classes.

## Evidence rule

Every rank and checkpoint needs student evidence. Evidence can be short. A strong one-sentence explanation is better than a long worksheet that hides misunderstanding.

## Dignity rule

Do not call students low-level readers. Do not make path ranks public in a way that creates comparison. The public story is training and growth. The private record is assessment and next-step planning.
