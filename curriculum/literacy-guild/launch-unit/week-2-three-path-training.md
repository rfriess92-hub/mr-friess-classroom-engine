# Week 2 Three-Path Training

## Purpose

Week 2 teaches students how the Literacy Guild actually works after launch.

The goal is not to start a full Voice Trial yet. The goal is to help students practise all three paths, use the first ability cards, try simple party roles, and submit clean solo evidence.

## Main goals

Students will:

- understand the difference between Voice, Insight, and Wordcraft
- set up or refine the character sheet
- use one ability card from each path
- practise one party role
- complete one short solo evidence task
- log evidence in the Guild Passport

## Teacher stance

Keep the mechanics light. Teach the reading moves more than the game terminology.

By the end of Week 2, students should know the routine better than the lore.

## Core student message

This week, you will try all three training paths. You are not choosing a permanent identity. You are learning which strategies help you read better.

## Materials needed

- Guild Passport
- character sheet
- ability cards: Echo Cast, Gist Map, Context Torch
- party role cards
- short shared text or text set
- teacher tracker
- Game Master reference card

## Ability cards introduced

Start with only three abilities.

| Path | Ability | Strategy |
|---|---|---|
| Voice | Echo Cast | Repeat a model read to build fluency. |
| Insight | Gist Map | Say what a section is mostly about. |
| Wordcraft | Context Torch | Use nearby words to infer meaning. |

Do not introduce all ability cards yet.

## Party roles introduced

Start with three roles.

| Role | Use |
|---|---|
| Quest Captain | Keeps the group on task. |
| Insight Keeper | Checks gist and meaning. |
| Wordsmith | Tracks key words and context clues. |

Add Voicekeeper, Evidence Hunter, and Support Checker after routines are smoother.

## Day 1: Character and path reset

### Goals

- review the three paths
- introduce the character sheet
- connect character identity to training, not reading level

### Flow

1. Daily Encounter: Which path does this task use?
2. Review Voice, Insight, and Wordcraft.
3. Introduce the character sheet.
4. Students choose or confirm starting class.
5. Students choose one personal training goal.
6. Character Log: record class choice and first goal.

### Evidence

- character sheet started
- training goal written

## Day 2: Voice training

### Goals

- introduce Echo Cast
- practise safe rereading
- build confidence with oral rehearsal

### Flow

1. Daily Encounter: punctuation and phrase chunking.
2. Game Master uses Strategy Reveal: Echo Cast.
3. Teacher models a short read.
4. Students echo read or partner rehearse a short passage.
5. Students mark one phrase that improved.
6. Character Log: Voice evidence note.

### Evidence

- marked phrase or reread note
- Voice reflection

## Day 3: Insight training

### Goals

- introduce Gist Map
- practise saying what a section is mostly about
- connect gist to evidence

### Flow

1. Daily Encounter: one-sentence summary choice.
2. Game Master uses Strategy Reveal: Gist Map.
3. Read a short section together.
4. Students write a first gist statement.
5. Party check: Does the gist match the text?
6. Students revise the gist.
7. Character Log: Insight evidence note.

### Evidence

- first gist
- revised gist

## Day 4: Wordcraft training

### Goals

- introduce Context Torch
- practise using nearby words to infer meaning
- build a small shared word bank

### Flow

1. Daily Encounter: unknown word in a sentence.
2. Game Master uses Strategy Reveal: Context Torch.
3. Students identify one useful or unknown word.
4. Students use nearby clues to make a smart guess.
5. Party check: Which clue helped most?
6. Character Log: Wordcraft evidence note.

### Evidence

- word choice
- context clue explanation

## Day 5: First three-path mini quest

### Goals

- combine all three paths
- practise party support without replacing solo evidence
- submit one clean solo evidence task

### Flow

1. Quest Call: short text challenge.
2. Voice: echo or reread selected lines.
3. Insight: write one gist statement.
4. Wordcraft: explain one context clue.
5. Solo Evidence: students submit one final response that uses at least two paths.
6. Character Log: record XP, evidence, and next goal.

### Evidence

Students submit one of the following:

- a revised gist plus one context clue explanation
- a fluency reflection plus one gist statement
- a short answer using one key word and one text detail

## XP guidance

Keep XP simple.

Suggested Week 2 awards:

| Action | XP |
|---|---:|
| Character sheet started or updated | 10 |
| Used first ability card correctly | 15 |
| Completed party role responsibly | 10 |
| Submitted solo evidence | 20 |
| Completed Character Log reflection | 10 |

Award in batches. Do not turn the week into accounting.

## Teacher tracker focus

Track only:

- who understands the three paths
- who submitted solo evidence
- which students need teacher-table support
- which ability cards were used correctly
- who needs help logging evidence

## End-of-week teacher check

Ask:

- Can students tell Voice, Insight, and Wordcraft apart?
- Can students use the Passport without too much teacher time?
- Did party roles help or distract?
- Did any game mechanic create confusion or comparison?
- Which path is ready for a deeper focus next?

## Adjustment rule

If Week 2 feels rushed, repeat Three-Path Training for another week before starting a path arc.

If Week 2 runs smoothly, the next arc can begin with Voice Trial because students will already understand that Voice is one part of the larger system.
