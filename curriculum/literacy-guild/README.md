# Literacy Guild

The Literacy Guild is the first course skin for the reusable Guild Engine.

## Anchors

Teacher-facing:

- Fluency
- Comprehension
- Vocabulary

Student-facing:

- Voice
- Insight
- Wordcraft

## Promise

Students train reading skills, collect evidence, reflect on growth, and set next goals. They are not ranked against one another.

## Versions

This folder will support two class versions:

1. A full-year Grade 8 version.
2. A semester version for a returning cohort.

## Shared loop

**Train -> Attempt -> Feedback -> Revise -> Log Evidence -> Level Up**
