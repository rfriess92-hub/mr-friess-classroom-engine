# Guild Engine Build Timeline

Goal: have a classroom-ready version before the start of school.

## Build order

1. Design brief
2. Core gamification engine
3. Literacy path model
4. Assessment and feedback layer
5. Daily routine
6. Weekly quest structure
7. Two-track implementation plan
8. Student materials
9. Teacher tools
10. Launch unit
11. QA and simplification
12. Final assembly

## Working principle

Build the reusable game engine first. Then build the Literacy Guild as the first course skin. Add lesson content only after the model is stable.

## Launch-ready means

The launch version needs the core rules, student materials, teacher trackers, first quest, first boss task, and first weeks of classroom routines. It does not need the entire year of content built in advance.
