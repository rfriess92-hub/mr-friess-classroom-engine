# Guild Engine

The Guild Engine is the reusable gamification model for classroom systems in this repository.

It is not a course by itself. It is a portable structure that can be adapted to literacy intervention, English, Social Studies, Science, ADST, Career Education, or other classes.

## Core idea

Students grow through a repeatable loop:

**Train -> Attempt -> Feedback -> Revise -> Log Evidence -> Level Up**

The system rewards growth behaviours, strategy use, evidence, revision, reflection, and sustained practice. It does not reward public ranking, raw achievement alone, or comparison between students.

## Reusable pieces

- Quests
- Training tasks
- Evidence logs
- XP
- Levels
- Rank-ups
- Boss tasks
- Private progress checks
- Reflection checkpoints
- Class/team goals
- Teacher conferences

## Course skins

Each class can use the same engine with different learning paths.

Examples:

- Literacy Guild: Voice, Insight, Wordcraft
- Science Guild: Observe, Explain, Apply
- Social Studies Guild: Evidence, Perspective, Cause and Consequence
- Writing Guild: Ideas, Structure, Style

## Current implementation

The first implementation is `curriculum/literacy-guild/`, designed for Grade 8-10 reading intervention students significantly below grade level.
