# Literacy Intervention Tracker

A separate intervention product. It is not a graded ELA course tracker and does not calculate a global literacy percentage.

## Stable fallback

Archive member: `Mr_Friess_Literacy_Intervention_Tracker_V2.1.1.xlsx`

V2.1.1 preserves the validated four-tab workflow and remains available while V3 is piloted.

## V3 beta

Archive member: `Mr_Friess_Literacy_Intervention_Tracker_V3.0_Beta.xlsx`

V3 adds:

- stable student IDs;
- controlled skill and measure registries;
- metric-aware accuracy, WCPM, rubric, count, rate, and duration progress;
- chronological evidence series;
- selected-student progress view and chart;
- flexible-group planning and group-fit checks;
- family-safe communication bridge.

## Working tabs

1. Hub
2. Sessions
3. Assessment & Mastery
4. Student Log

## Communication bridge

V3 exposes the standardized bridge at:

```text
1 Hub!BF4:BY39
```

Literacy records leave course-grade fields blank and identify the source as `literacy_intervention`.

Both workbook files are packaged in `../releases/2026-07-19/Mr_Friess_Tracker_Patches_Literacy_V3_and_Email_Bridge.zip`.

## Pilot status

V3 is a beta. Keep V2.1.1 available until Google Sheets import, classroom navigation, progress calculations, grouping, and communication preview have been piloted.
